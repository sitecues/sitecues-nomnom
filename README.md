## Web server ##
Default port is 3001, but can be set via PORT environment variable.
Do NOT prefix options with --
```
# Launch server
node bin/nomnom-server
or
PORT=3123 node bin/nomnom-server

# Use web service as follows:
localhost:3001?[options]

# Example:
localhost:3001?...
```
